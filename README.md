# Panduan Membeli VPS di Hostinger

Dokumen ini menjelaskan langkah demi langkah cara membeli VPS (Virtual Private Server) di Hostinger, mulai dari persiapan sampai pembayaran. Ditulis singkat dan rapi.

---

## 1. Membuat Akun Hostinger (Jika Belum Punya)

1. Buka situs: **[hostinger.com](https://www.hostinger.com)**.
2. Klik tombol **Sign up** / **Get started**.
3. Isi:
   - Email
   - Password
4. Verifikasi email jika diminta.

Akun ini digunakan untuk mengelola semua produk (domain, shared hosting, VPS, dll) lewat **hPanel**, panel milik Hostinger.

---

## 2. Cara Membeli VPS di Hostinger (Pengguna Baru)

### 2.1 Masuk ke Halaman VPS

1. Login ke akun Hostinger.
2. Arahkan ke menu:
   - **Hosting â†’ VPS** atau
   - Buka langsung halaman **VPS Hosting** dari menu pricing/produk.

### 2.2 Pilih Paket VPS

Di halaman VPS, kamu akan melihat beberapa paket (misal **KVM1**, **KVM2**, dst.) dengan kombinasi:

- Jumlah vCPU
- RAM
- Storage NVMe
- Bandwidth/network speed (biasanya sampai 1 Gbps)

Langkah memilih:

1. Bandingkan spesifikasi dengan kebutuhan aplikasi.
2. Klik tombol **Select** / **Add to cart** pada paket yang diinginkan.

### 2.3 Pilih Periode Berlangganan

Hostinger biasanya memberi beberapa opsi lama kontrak:

- 1 bulan
- 12 bulan
- 24 bulan, dst.

Semakin panjang masa berlangganan, biasanya diskon makin besar.

Pada bagian ini:

1. Pilih durasi (misal **12 bulan** / **24 bulan**).
2. Jika ada kode kupon, masukkan pada kolom **Coupon code**.

### 2.4 Buat / Konfirmasi Akun

Jika kamu belum login, Hostinger akan meminta:

- Login ke akun **existing**, atau
- Membuat akun baru (email + password).

Jika sudah login, langkah ini bisa dilewati otomatis.

### 2.5 Masukkan Detail Pembayaran

1. Isi **billing details** (nama, alamat, negara).
2. Pilih metode pembayaran yang tersedia.
3. Konfirmasi total biaya:
   - Harga paket VPS
   - Diskon (jika ada)
   - Pajak / PPN (jika berlaku)
4. Klik **Pay now** / **Submit payment**.

Setelah pembayaran berhasil, VPS akan masuk ke akunmu dan bisa diakses via **hPanel** di bagian **VPS**.
